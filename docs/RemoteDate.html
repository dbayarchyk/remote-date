dateNow() → {Date}

Returns the newly-created Date object represents the current date and time as of the time of instantiation. The returned date's timestamp is the same as the number returned by remoteDate.now(). The Remote instance needs to be initialized once first via a constructor or remoteDate.setRemoteTime() before being able to use remoteDate.dateNow().

now() → {number}

Returns the number of milliseconds elapsed since the epoch, which is defined as the midnight at the beginning of January 1, 1970, UTC. The Remote instance needs to be initialized once first via a constructor or remoteDate.setRemoteTime() before being able to use remoteDate.now().
